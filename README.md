# YouTube-clone

A simple YOUTUBE front-end clone of a video platform — built with plain HTML and CSS.  
This project demonstrates layout, thumbnail grids, and responsive design styling.


## Demo

Open `code.html` in your browser to view the project locally.


## Features

-> Grid of video cards / thumbnails
-> Header/navigation layout
-> Profile thumbnails (in `profiles/`)
-> Clean HTML/CSS structure — easy to extend


## Tech stack

- HTML
- CSS
